# EONParser - Log Analysis Tool

**EONParser** is an interactive desktop application built using PyQt5 that allows users to upload, analyze, visualize, and generate reports from log files using natural language queries. It combines log parsing, NLP, analytics, and visual reporting into a single tool for security and network professionals.
